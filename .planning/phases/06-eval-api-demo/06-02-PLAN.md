## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Tampering | POST /ask body | mitigate | Pydantic AskRequest validates `question: str` min_length=1 at route entry; FastAPI returns 422 on malformed/empty body |
| T-06-02 | Information Disclosure | /ask error responses | mitigate | Handler returns only report+findings; do not echo stack traces or secrets. GEMINI_API_KEY never returned/logged |
| T-06-03 | Denial of Service | /ask (each call runs full agent + LLM) | accept | Single-user Labs demo, no public exposure; rate-limiting out of scope for portfolio v1 (documented) |
| T-06-04 | Tampering (SQL injection) | question → sql_tool generated SQL | accept | Pre-existing mitigation: sql_tool validates generated SQL via EXPLAIN against the real schema before exec (TOOL-01, Phase 2). No new surface added here |
</threat_model>
