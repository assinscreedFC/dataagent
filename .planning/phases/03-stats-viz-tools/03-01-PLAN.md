State shape (src/dataagent/agent/state.py):
```python
class AgentState(TypedDict):
    question: str
    plan: list[str]
    findings: Annotated[list[dict], add]   # reducer: returned findings are APPENDED
    messages: Annotated[list, add_messages]
    iterations: int
    max_iterations: int
    db: Annotated[DuckDBPyConnection, UntrackedValue(...)]
    report: str
```

Node contract (from nodes.py): a node is `def node(state: AgentState) -> dict`.
Returning `{"findings": [f1, f2]}` APPENDS those findings via the `add` reducer.
Existing sql_tool findings have shape (numeric data lives in `rows` + `columns`):
```python
{"source": "sql_tool", "subquestion": str, "sql": str,
 "tables": [...], "rows": [(...), ...], "columns": [...], "attempts": int}
# OR on error: {"source": "sql_tool", "subquestion": str, "sql": str, "error": str, "attempts": int}
```

Polars correlation API (queries.py uses Polars throughout):
```python
import polars as pl
df.select(pl.corr("col_a", "col_b")).item()   # Pearson coefficient, float
```
